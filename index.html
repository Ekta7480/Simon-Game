🎮 Simon Says - Color Pattern

Press any key or click Start to begin

High Score: Level 0

🕹️ How to Play:

  • Pattern: 3 blocks are blue, 1 block is red
  • Watch: The computer flashes a sequence
  • Repeat: Click the blocks in the same order
  • Level Up: Each level adds one more step
  • Challenge: Remember longer sequences!

🎯 Game Flow:

Level 1: red

Level 2: red → blue

Level 3: red → blue → blue

And so on... The sequence gets longer!
